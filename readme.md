# Customer Lifetime Value Analysis

This project analyzes customer purchasing behavior to estimate Customer Lifetime Value (CLV) and identify high-value customers.

## Tools Used
Python
SQL
Power BI
Pandas
Matplotlib

## Project Structure

data – dataset  
notebooks – data analysis  
sql – SQL queries  
visuals – charts  
dashboard – Power BI dashboard  
reports – project report  

## Key Insights

• Identified high-value customers
• Analyzed revenue trends
• Evaluated purchase frequency
• Built an interactive Power BI dashboard
